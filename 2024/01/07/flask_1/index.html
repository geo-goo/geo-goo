介绍 Flask

Flask 是一个基于 Python 的微型 Web 框架。它由 Armin Ronacher 开发，目的是创建一个简单但可扩展的 Web 框架。Flask 以其灵活性和简单性著称，适合初学者学习和构建小型到中型的 Web 应用程序。Flask 遵循 WSGI 标准，并且使用 Jinja2 模板引擎。

Flask 的主要特点：

  1. 轻量级：没有依赖过多的外部库，核心功能精简。
  2. 模块化：可以根据需求添加各种扩展（如数据库、表单处理、认证等）。
  3. 易于学习：具有简单直观的 API，适合初学者。
  4. 灵活性高：允许开发者根据需求进行高度定制。
